# Phase 1 受け入れチェックリスト

## 前提

- 対象 OS: Windows 11。
- 対象ディスプレイ: JAPANNEXT `JN-MD-IPS7842`。
- 実装技術: WPF / .NET 8。

## ローカル起動

- [ ] `.NET SDK 8` がインストールされている。
- [ ] `dotnet run --project src/Minimoniclock/Minimoniclock.csproj` で起動できる。
- [ ] ネットワーク接続なしで起動できる。

## 時計表示

- [ ] 時刻が `HH:mm` の 24 時間表記で表示される。
- [ ] 1 分経過後に表示が更新される。
- [ ] 日付またぎで時刻表示が破綻しない。

## 縦向き JUSTFIT

- [ ] `400x1280` 相当のウィンドウで時刻が欠けない。
- [ ] 縦向きで余白が過剰にならない。
- [ ] DPI 100% で主要表示が欠けない。
- [ ] DPI 125% で主要表示が欠けない。
- [ ] DPI 150% で主要表示が欠けない。

## 横向き JUSTFIT

- [ ] `1280x400` 相当のウィンドウで時刻が欠けない。
- [ ] 横向きで余白が過剰にならない。
- [ ] DPI 100% で主要表示が欠けない。
- [ ] DPI 125% で主要表示が欠けない。
- [ ] DPI 150% で主要表示が欠けない。

## 常時表示

- [ ] `FIT` でボーダーレス最大化できる。
- [ ] `FIT` を再度押すと通常ウィンドウへ戻る。
- [ ] `TOP` で常に手前表示を切り替えられる。

## データ・通信

- [ ] ポモドーロ、アラーム、履歴、ログを保存しない。
- [ ] アプリ独自のログファイルを生成しない。
- [ ] 外部通信を行わない。
